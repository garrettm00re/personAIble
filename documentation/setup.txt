 Clear setup instructions - make it so anyone can follow in your footsteps

ensure you have python 3.12 installed

copy over all files/folders in the root directory from https://github.com/garrettm00re/personAIble

put API keys in .env

install dependencies: pip install -r requirements.txt

Create the application data:
    A user, fake in this case, answers some questions yielding charlesRiverInterview.txt (no action required)
    executing charlesRiverAssets/charlesRiver.ipynb creates pertinent json files in charlesRiverAssets/

Run python app.py to start the development server
or
Deploy to vercel:
    execute vercel in the root directory (make sure you have the vercel cli installed)
